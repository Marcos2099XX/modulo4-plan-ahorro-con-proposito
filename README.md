# modulo4-plan-ahorro-con-proposito
Actividad del Módulo 4 "Ahorra con propósito": página web donde el estudiante toma su meta SMART prioritaria y diseña un plan de ahorro básico por periodos (meses o semanas), definiendo cuánto ahorrar, de dónde saldrá ese dinero en su presupuesto y cómo enfrentará posibles obstáculos para mantenerse en el plan.
